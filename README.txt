Ficheiros de input

- Os ficheiros de input utilizados encontram-se na pasta Mapas e correspondem, essencialmente, aos ficheiros com as informações sobre
os nós e sobre as arestas de cada um dos mapas utilizados. Contém ainda, para cada mapa, um ficheiro de tags para se guardas os 
vértices onde se localizam os restaurantes. 

- Na pasta da cidade de penafiel, existem ainda 2 ficheiros que servem para tornar mais rápido o processo de carregar o grafo desta
cidade, guardando-se as distâncias entre os vértices nestes ficheiros.
